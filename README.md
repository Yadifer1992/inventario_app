# Sistema de Gestión de Inventarios

## Descripción
Este proyecto es un **sistema de gestión de inventarios** desarrollado en **Python**.  
Permite añadir, actualizar, eliminar y buscar productos en un inventario de una tienda usando **Programación Orientada a Objetos (POO)** y un menú interactivo en la consola.

---

## Funcionalidades
- Añadir productos con ID, nombre, cantidad y precio.  
- Actualizar cantidad o precio de productos existentes.  
- Eliminar productos por su ID.  
- Buscar productos por nombre (búsqueda parcial y flexible).  
- Mostrar todos los productos en formato de lista.
